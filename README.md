# AI水军评论检测与溯源系统

## 项目简介
基于AI的水军评论检测与溯源系统前端界面，实现评论数据管理、AIGC内容检测、水军群体溯源分析和可视化展示。

## 新增功能
### 完整的登录注册系统
- 用户注册和登录功能
- 三种用户角色：管理员、分析师、访客
- 密码显示/隐藏切换
- 记住登录状态
- 忘记密码功能
- 用户个人信息管理
- 退出登录功能

### 演示账户
- **管理员**: admin / admin123
- **分析师**: analyst / analyst123  
- **访客**: visitor / visitor123

## 功能模块
### 仪表盘
- 核心数据统计和图表展示
- 实时数据监控
- 最近活动记录

### 数据管理
- 评论数据导入、查询、维护
- 多平台数据支持
- 数据筛选和分页
- 数据导出功能

### AI检测分析
- AIGC概率预测
- 水军集群发现
- 批量检测和实时监测
- 详细检测结果展示

### 可视化分析
- 水军网络关系图
- 时间线分析
- 热力图分析
- 证据链查看

### 系统管理（管理员权限）
- 用户管理
- 任务管理
- 系统设置
- 操作日志

## 技术栈
- HTML5 + CSS3 + JavaScript
- Vue.js 3.x（CDN方式）
- Chart.js 数据可视化
- Font Awesome 图标库
- 本地存储（LocalStorage）用于用户状态管理

## 运行方式
1. 使用VS Code打开项目
2. 安装Live Server扩展
3. 右键点击index.html → "Open with Live Server"
4. 浏览器访问 http://localhost:5500
